## Overview
The Smart Waste Management Voice Bot is a multi-agent conversational AI solution designet to automate and streamline waste management services for municipalities, smart cities, residential communities, and waste collection providers.

Built using Google CX Agent Studio, the system leverages a Root Agent and multiple specialized Sub-Agents to efficiently handle citizen requests, reduce manual intervention, and service delivery.
